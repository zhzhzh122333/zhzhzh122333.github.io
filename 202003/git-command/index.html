报错总结

1、提交文件时报以下错误：

fatal: Will not add file alias ‘themes\next\languages\zh-HK.yml’ (‘themes\next\languages\zh-hk.yml’ already exists in index)

原因分析： 提交的文件修改了文件名，并且只改了大小写，由于git默认不区分大小写，因此提交的时候会报错。
解决方案： 先将文件改成其他名称，然后改回最终的名称提交。
