CentOS下安装Sqoop

发表于 2016-08-31 更新于 2020-03-07 分类于 Linux

系统：CentOS 6.5桌面版
版本：Sqoop 1.4.6

下载并解压

下载Sqoop，这里下载的版本是sqoop-1.4.6.bin__hadoop-2.0.4-alpha.tar.gz

配置环境变量

编辑/etc/profile，加入sqoop的环境变量

编辑配置文件

编辑sqoop目录下的配置文件

加入相应的目录，注意hadoop-common和hadoop-mapred都是填hadoop的安装目录

添加数据库驱动

根据你的数据库，下载相应的驱动jar包放入sqoop/lib目录下，这里用的是sqlserver
附上下载连接sqlserver
