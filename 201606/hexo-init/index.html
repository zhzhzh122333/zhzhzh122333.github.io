安装Node.js

Node.js

安装git

git
把git安装目录下的cmd文件夹路径添加到系统环境变量(windows)

安装cnpm

Node.js自带的npm容易被墙，安装cnpm代替npm

npm install -g cnpm --registry=https://registry.npm.taobao.org

安装Hexo

cnpm install -g hexo-cli

初始化网站目录

hexo init {Your_Folder}

开启网页测试

cd {Your_Folder}
hexo server

访问http://localhost:4000/
若页面能正常显示，则安装成功
